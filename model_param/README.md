# Model Checkpoints Placeholder

该目录用于保存训练得到的模型权重和序列化结果，公开仓库不附带这些文件。

常见输出文件名包括：

- `baseline_model.pkl`
- `resnet_model.pkl`
- `finetuned_resnet_model.pkl`
- `profit_resnet_model.pkl`
- `lightgbm_model.pkl`
- `xgboost_model.pkl`
- `ft_transformer_model.pkl`
- `dnn_model.pkl`
- `pretrained_encoder.pth`

训练脚本会自动在本地生成这些文件。
